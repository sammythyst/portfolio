# portfolio
Module 2 Challenge - Advanced CSS


The purpose of this project is to start and continually add to my personal portfolio for future employers to use to understand my capabilities as a web developer.

To organize my work, it will be my goal to place my most recent project larger at the top, followed by four of my less recent works, smaller than the first. When my projects list grows beyond the five examples, I will plan to add a "see more" option. This will direct viewers to a separate page showing all of my works, keeping most recent towards the top, as to not overcrowd my main page.
